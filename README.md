# customize-rime-cloverpinyinyin
在🍀️四叶草拼音输入方案的基础上进行定制:
- 增加了英语的输入;
- 增加emoji_catagory中的简体中文对应的符号；
- 增加了一些其他输入法词库；
- 皮肤默认为mac绿色，可以在输入法设定里面选择其他的皮肤；
- 删除了其他输入法，有需要可以自己在加进去；
